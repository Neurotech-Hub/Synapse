Recommended CSV flow

Upload first with Commit valid rows unchecked. Review row-level errors, then rerun with commit enabled once the dry run looks right.

Duplicate handling

Duplicates are detected by external_id first, then normalized source URL. Leave update-on-duplicate off unless you intentionally want the CSV to overwrite existing rows.

Public safety

Imported records remain private until you manually run OpenAI generation. A successful generation marks the row reviewed/public; sparse results stay private and ask for more context.
