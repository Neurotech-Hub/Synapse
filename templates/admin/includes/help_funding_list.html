Funding records

Use this page to review imported or manually created funding opportunities. Public visibility is separate from status: a record must be reviewed and public before it can appear on Funding Radar.

Effort index

Effort describes likely application burden, not value. Mild, moderate, heavy, and unknown are intentionally simple labels that can be overridden on the detail page.

CSV import

Start with Import CSV for batch entry. Dry-run validation catches missing titles, bad URLs, invalid labels, and duplicates before committing rows.
