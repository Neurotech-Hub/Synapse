Places

Each place is a named point (latitude / longitude) optionally linked to an organization. Use View to edit fields, see the map, and refresh the place-level rollup persona.

Columns

  • Slug — stable URL key (derived from display name when you save).
  • Name — place label shown in admin and prompts.
  • Lat / Lng — coordinates for the embedded map and rollup context.
  • Organization — optional parent org for catalog and persona rollups.
  • Rollup — persona snapshot build status for this place (same JSON shape as people/orgs).
