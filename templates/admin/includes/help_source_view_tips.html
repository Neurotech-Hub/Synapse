Approval & settings

Approve for polling clears pending first; use the optional radios below it if you also need to attach a person or organization owner when approving. Disapprove sends it back to the review queue. Hide from polling skips polling without deleting the source.

Source owner

Assign either a person or an organization (not both). Ownership drives persona paths for lead candidates and whether the source is in the Hub corpus organization selected under Leads → Hub settings.

Snapshots

For html_page sources, each poll stores a SHA-256 of the fetched HTML below. Each new snapshot usually creates a companion content item (title/snippet from visible text, with Ollama when enabled). The Sources list opens the same snapshot table as a snapshots-only page.

Content items

RSS: one row per feed entry. html_page: one row per new snapshot hash (snippet may use Ollama).
