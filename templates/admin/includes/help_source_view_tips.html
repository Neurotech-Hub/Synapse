Approval & settings

Approval controls whether a source is included in Poll now. Disapprove sends it back to the review queue. Hide from polling skips polling without deleting the source.

Tracked entities

Tags feed into the lead qualification prompt (entity catalog). Tag the same subject on Hub and world feeds when it helps the model resolve duplicates. Create entities under Entities if the list is empty.

Snapshots

For html_page sources, each poll stores the raw HTML fingerprint here. Each new snapshot gets a companion content item (title/snippet from visible text, with Ollama when enabled).

Content items

RSS: one row per feed entry. html_page: one row per new snapshot hash (snippet may use Ollama).
