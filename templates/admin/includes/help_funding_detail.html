Funding workflow

Import rows from CSV, then manually run Generate public-ready card with OpenAI. If generation succeeds, Synapse applies the public fields and marks the row reviewed/public by default.

Maintenance

The maintenance card keeps lower-level controls visible: edit manually, fetch source text, recalculate effort, publish, or archive. These are secondary to the OpenAI generation path.

Public boundary

Public Funding Radar can show title, sponsor, amount, deadline, effort, tags, public summary, and the official source.
