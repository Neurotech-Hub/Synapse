URL & kind

  • Label is optional—it appears in the Sources list so non-obvious URLs stay scannable.
  • Use Visit URL (after you type a URL) to confirm the link opens.
  • RSS feed — one content item per feed entry.
  • HTML page — we store a SHA-256 of the page each poll; a new hash creates a new content item (change detection).

Polling

  • Hide from polling keeps the source but skips it on Poll now.
  • Sources waiting for approval are excluded from polls until approved.

Entities & Hub

After you save, tag Neurotech Hub on the source edit page — that ingest then counts toward the Hub corpus snippets used when you run lead candidates under Leads. Prospect feeds can stay untagged if you prefer them excluded from Hub evidence.

Poll only ingests content items. Lead candidates queue from Leads (not poll).
