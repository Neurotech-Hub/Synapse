What matching does today

This dashboard runs deterministic funding-to-idea matching only. It uses tag and keyword overlap, funding status, effort, and evidence features; it does not call an LLM.

Review states

New edges are private and need review. Accept or reject each edge before relying on it. Public pages only use accepted edges marked public_safe or public.

Hypotheses

The Hypothesis action creates a private draft from one funding-to-idea edge. Rich target-centered collaboration generation remains future work.
