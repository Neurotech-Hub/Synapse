Sources list

Use By owner for person-owned, organization-owned, and unassigned sections. Switch to By website to see one list grouped by hostname (site rows with counts, then rows for that origin).

Columns

  • Label — optional admin title for the ingest (set on Edit source); use when the canonical URL alone is obscure.
  • Hub corpus — yes when this source is in the corpus of the organization set under Leads → Hub settings as the Hub corpus organization (those ingests supply Hub evidence for lead reports). Shows — if that setting is unset.
  • Items — opens the source page scrolled to content items.
  • View — opens the source: URL, polling, tracked entities, snapshots, content previews, approve/disapprove/hide—and Delete source when needed.
